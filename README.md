# 🛡️ Portfólio de Qualidade – Desafio QA 2026

Bem-vindo(a) ao meu repositório de testes do sistema **NaSalinha**.  
Este projeto tem como objetivo realizar uma auditoria completa de qualidade, documentando bugs, planejando casos de teste e validando funcionalidades essenciais da aplicação.

---

## 📌 Sobre o Projeto

**NaSalinha** é um sistema web voltado para check-in por foto, autenticação JWT e sistema de pontos.  
Este repositório contém toda a documentação e evidências de testes realizados durante o desafio de QA.

---

## 🧪 Tecnologias e Ferramentas

| Área                | Ferramenta                      |
|---------------------|----------------------------------|
| Testes de API       | Insomnia / Postman               |
| Testes de Interface | Navegador (Chrome/Firefox)       |
| Gestão de Testes    | GitHub Projects + Issues         |
| Documentação        | Markdown, PDF, Planilhas         |
| Ambiente            | Docker + Docker Compose          |

---

## 🚀 Como Rodar o Projeto Localmente

```bash
# 1. Clone o repositório

# 2. Acesse a pasta do projeto
